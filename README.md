# TeamCenter 1.0

Versione ufficiale pronta per GitHub Pages.

## Pubblicazione
Carica **tutti i file e le cartelle** nella root del repository GitHub. `index.html` deve restare nella root.

## Moduli inclusi
- Profilo società: nome, logo e colori
- Convocazioni: dati gara, 1–20, staff, anteprima, PNG e PDF
- Gestione match: timer con centesimi, recupero, risultato, goal, assist, cartellini e statistiche
- PWA: manifest e service worker
- Salvataggio locale del dispositivo

## Limiti della 1.0
Google Drive, storico online e autenticazione verranno collegati nelle versioni successive.
