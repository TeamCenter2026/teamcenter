TEAMCENTER 1.9.1 — CORREZIONE AVVIO, PROFILO E LOGO

PROBLEMA RISOLTO

La versione 1.9 poteva interrompere JavaScript prima del bootstrap perché alcuni
listener facevano riferimento a vecchi elementi HTML non più presenti.

Conseguenze:
- titolo TEAMCENTER invece di CSV Breda;
- logo assente;
- dati MASTER non applicati;
- moduli Allenamenti e Convocazioni non inizializzati correttamente.

MODIFICHE

- Nuovo file bootstrap.js.
- MASTER, logo e squadre vengono caricati prima di app.js.
- Titolo Home impostato sul nome società.
- Logo ufficiale applicato alla Home.
- Colori applicati dal foglio MASTER.
- Listener legacy resi sicuri.
- Ripristino corretto della schermata Allenamenti.
- Ripristino corretto della schermata Convocazioni.
- Cache aggiornata alla versione 1.9.1.
- Allenamenti e Convocazioni aggiunti agli asset del Service Worker.

INSTALLAZIONE

1. Carica su GitHub TUTTI i file della cartella teamcenter-main.
2. Commit consigliato:
   TeamCenter 1.9.1 - Fix bootstrap profilo e logo
3. Attendi GitHub Actions verde.
4. Apri:
   https://teamcenter2026.github.io/teamcenter/?v=191
5. Premi CTRL + F5.

PULIZIA CACHE, SE NECESSARIA

Chrome:
- F12
- Application
- Service Workers
- Unregister
- Storage
- Clear site data
- Ricarica con ?v=191

APPS SCRIPT

La logica backend non cambia.
È comunque incluso CODICE-GS-COMPLETO-1.9.1.txt.

Puoi sostituire Codice.gs e creare una nuova distribuzione, ma per la sola
correzione della Home non è indispensabile se il deployment 1.9 risponde già.

TEST

1. Apri la Home.
2. Controlla che compaia CSV BREDA.
3. Controlla il logo ufficiale.
4. Controlla i colori societari.
5. Apri Rosa, Staff, Allenamenti e Convocazioni.

7. Verifica che il modulo venga inizializzato correttamente.
